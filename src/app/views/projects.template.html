Teacher ToolBox

Teacher ToolBox is a Java desktop application created by myself and 5 classmates for our capstone project. It allows teachers to import rosters, add, edit, or delete student information, keeps track of attendance and grades, and runs a number of reports. We used JavaFX in the creation of the application. For the data storage we used SQL and a local database.

Homework Check-In

Homework Check-In is an Android application created for teachers to use in their classrooms. It allows the teacher to input their students names, which is saved to a local SQLite database, and then allows the students to check off their name when they turn in their homework. The teacher can then save each days list to a file that can then be loaded back into the app for later viewing.

Pretty In Pink Classic Website

Pretty in Pink Classic is a foundation focused on providing financial assistance to women who are uninsured or underinsured breast cancer patients. This website allows people to sign up for the annual fundraising event. I developed this website for them to use for sign ups in their yearly fundraiser. I used HTML, CSS, and Bootstrap for the frontend and PHP and SQL to handle the backend.
